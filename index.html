Term project that I'm working on

ProjectX

ProjectX is the code name for my project that I'm doing for myself and also as a part of my course CIS 641 at GVSU. It is basically a classifed advertisement posting website. The application would feature a simple interface to post the classifieds and would also provide users with a smart search feature that would serve them with precise and relevant information they're looking for.

Technically, this project shall be implemented using Java Spring Framework in the backend and Angular for the frontend. To maintain the contract for these, we would be following API-first apporach, preferrably with OpenAPI tools. We shall also use IAM/IDP for user management and access controls. For the search feature, we may implement ElasticSearch for real-time full-text search and the search analytics. The project shall be implemented using all modern principles of software development from TDD to CI/CD.
